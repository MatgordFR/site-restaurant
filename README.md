---

## ✨ Ce que ça montre

Un **one-pager de restauration** à l'identité forte, pensé pour donner faim :

- **Une ambiance « braise »** — fond sombre chaleureux, four à bois vivant (flammes et halo qui vacillent en CSS/SVG).
- **Une cuisson interactive** — faites glisser, ou laissez le four faire : la Margherita cuit en 90 s, le bord se tache de « léopard », la mozzarella dore sous vos yeux.
- **Une carte éditoriale** — colonnes *Rosse* / *Bianche*, leaders pointillés, descriptions appétissantes.
- **Les repères d'un vrai artisan** — 485°C, 90 s de cuisson, 72 h de maturation, produits DOP.

## 🎨 Le craft

- **Identité dérivée du sujet** — palette « feu de bois », rien d'un gabarit générique.
- **Polices auto-hébergées** — Bricolage Grotesque (titres) + Fraunces (corps & carte). Aucune requête tierce.
- **Graphismes dessinés à la main** — four, icônes et pizza en SVG.
- **Accessible & responsive** — navigation clavier, focus visible, `prefers-reduced-motion` respecté, du grand écran au mobile sans débordement.

## 📁 Structure

```
index.html            → la page
assets/css/forno.css  → design system + identité braise
assets/js/forno.js    → reveals + cuisson interactive
assets/fonts/         → Bricolage + Fraunces (auto-hébergées)
```

## 🚀 Lancer en local

```bash
python3 -m http.server 8000
# puis http://localhost:8000
```
